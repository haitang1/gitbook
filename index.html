金特尔商城后台管理系统操作手册

[!NOTE|labelVisibility:hidden|iconVisibility:hidden] 本操作手册由Markdowm编写而成

使用教程

安装node.js

  • 使用nvm安装node,下载 nvm-windows最新安装包，直接安装即可。

下载指定的node.js版本

nvm install 12.17.0

查看下载的版本

nvm ls

使用相关版本

nvm use 12.17.0

查看node版本

node -v

查看npm版本

npm -v
  • npm加速

使用nrm管理npm镜像源

npm install -g nrm

查看能使用的镜像源

nrm ls

  npm -------- https://registry.npmjs.org/
  yarn ------- https://registry.yarnpkg.com/
* cnpm ------- http://r.cnpmjs.org/
  taobao ----- https://registry.npm.taobao.org/
  nj --------- https://registry.nodejitsu.com/
  npmMirror -- https://skimdb.npmjs.com/registry/
  edunpm ----- http://registry.enpmjs.org/

使用cnpm或taobao镜像

nrm use cnpm

[!WARNING] 使用nrm可能遇到的问题

找到报错的文件的路径

找到

const NRMRC = path.join(process.env.HOME, '.nrmrc');

改成

const NRMRC = path.join(process.env.USERPROFILE, '.nrmrc');

安装gitbook

使用npm下载

npm install gitbook-cli -g

查看npm版本

gitbook -V

生成目录

$ gitbook init
warn: no summary file in this book
info: create README.md
info: create SUMMARY.md
info: initialization is finished

使用 $ gitbook serve 命令,然后在浏览器地址栏中输入 http://localhost:4000 便可预览书籍。

使用 --lrport , --port 更改监听端口

gitbook --lrport 35731 --port 4002 serve
